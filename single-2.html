Descripción

Plataforma web sobre películas desarrollada de manera individual como proyecto académico. El sitio fue construido utilizando JSON, una API pública y una API propia basada en WordPress para gestionar contenido dinámicamente.

Mi Rol

Me encargué del desarrollo completo del proyecto, incluyendo la integración de datos, estructura del sitio y conexión con WordPress.

Decisiones Técnicas

Implementé distintos métodos de consumo de datos para comparar su funcionamiento y utilicé WordPress como API propia para permitir agregar, editar y eliminar contenido de forma dinámica.

Lo que Aprendí

Este proyecto me permitió entender en profundidad cómo funcionan los distintos métodos de consumo de datos y cuándo conviene usar cada uno. El manejo de tokens y llaves de autenticación para conectar con la API pública fue el punto más desafiante — entender el flujo de autorización y cómo proteger las credenciales me dio una base sólida que aplico en cada proyecto nuevo.
